## Quick start 🏄

Prerequisites: [Node (v19 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/chrisarevalo11/NatureLink-Vite.git
```

> Install dependencies:

```
yarn
```

> Run the project:

```
yarn dev
```
